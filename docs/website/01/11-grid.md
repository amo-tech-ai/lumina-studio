# Figma Make AI — Ecommerce Services Grid (Premium Editorial Style)

Create a clean, modern, editorial-style **3-column service grid** inspired by luxury fashion and ecommerce studios.

This section should feel premium, minimal, and content-driven with strong typography.

---

## 1. SECTION STRUCTURE  
Create a full-width section titled:

**“Our Ecommerce Services.”**

Below the heading, generate a **3-column grid** of 6 service cards.

Spacing rules:  
- Large vertical spacing between heading → grid  
- 32–40px gap between cards horizontally  
- 48–60px vertical spacing between rows  
- Cards aligned to a consistent baseline grid  
- Clean breathing room surrounding the entire block

Grid should be responsive:
- Desktop: 3 columns  
- Tablet: 2 columns  
- Mobile: 1 column stacked  

---

## 2. CARD COMPONENT DESIGN  
Each service card must contain:

### A. Image Placeholder  
- Ratio: 4:5 or 3:4  
- Soft shadow (very subtle)  
- Rounded corners: 8px  
- Label: “Service Image Placeholder”

### B. Title (H3)  
- Bold, editorial, serif or strong sans-serif  
- Example titles:  
  - **Packshot Photography.**  
  - **On-Model Fashion.**  
  - **Creative | Still Life**  
  - **Videography.**  
  - **eCommerce Apparel.**  
  - **eCommerce Jewellery.**

### C. Body Text  
- Short description (2–4 lines)  
- Clean sans-serif (Inter)  
- Comfortable line-height  
- Dark gray #333 text for readability  

### D. Auto Layout  
Enable auto layout for each card:
- Vertical padding: 24–32px  
- Spacing between elements: 12–16px  

Make each card a component so additional variations can be created later.

---

## 3. VISUAL STYLE  
### Color Palette  
- Background: soft off-white (#F7F5F2)  
- Card background: pure white (#FFFFFF)  
- Headings: black  
- Text: dark gray (#444)  
- Shadows: subtle ambient blur  

### Typography  
- Headings: Playfair Display or similar editorial serif  
- Body: Inter  
- Strong hierarchy — titles must feel bold and confident  
- Tight, clean spacing between title + body text  

### Imagery Style Notes  
Use placeholders that match:
- Fashion product photography  
- Editorial apparel shots  
- Accessories  
- Creative still life  
- Jewellery close-ups  
- Videography thumbnails  

---

## 4. INTERACTION & MOTION NOTES  
Add motion specifications to the frames (annotations only):

- **Hover:**  
  - Card lifts 4px  
  - Slight shadow deepen  
  - Image zooms from 1.0 → 1.03  
- **Scroll Reveal:**  
  - Fade-in + 20px upward motion  
  - Staggered animation: 120ms delay per card  

Figma Make AI should **document interactions inside component descriptions**.

---

## 5. RESPONSIVE BREAKPOINTS  
### Tablet  
- Auto-layout reflows into 2 columns  
- Slightly reduce card padding  

### Mobile  
- 1 column stack  
- Increase vertical spacing between cards  
- Image ratio stays consistent  

---

## 6. FINAL OUTPUT  
Deliver:  
- Desktop layout  
- Tablet layout  
- Mobile layout  
- Component set for cards  
- Layer groups properly named (Services → Card → Image / Title / Text)  

The final result must feel:  
**Premium, minimal, editorial, clean, fashion-forward, high-end ecommerce UI.**
