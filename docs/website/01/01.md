  ```ts

  ```

* Strict TypeScript settings
* No importmaps
* No loading files from external URLs

* Only:


# ✅ 2. Self-Audit the Entire Codebase

After generating all files, perform a **full audit**:

### 🔎 Check for:

#### Import errors

* Missing imports
* Wrong paths (`@/...` must resolve to `src/...`)
* No jsDelivr-style resolution
* No remote URLs for modules

#### TypeScript errors

* Wrong component props
* Missing children types
* Wrong interface definitions
* Router mismatch

#### Structural errors

* BrowserRouter declared more than once
* Layout used incorrectly
* File names not matching imports
* Missing exports

#### Build-time errors

* Anything that will break `vite build`
* Wrong script tags in index.html
* Incorrect Tailwind setup

#### Bad patterns to eliminate

* IMPORTMAPS
* TAILWIND CDN
* REMOTE SCRIPT LOADING
* Any reference to `https://ai.studio/src/main.tsx`
* Any reference to CDN resolver (`jsDelivr`)

You MUST identify AND FIX every issue.
