# Research Task: Build a Social Media Image Intelligence Knowledge Base

## Objective

Research and build a comprehensive knowledge base of social media image specifications that will power AI agents inside FashionOS/iPix.

The AI should recommend the best image types, sizes, and creative assets for brands based on campaign goals, industry, and platform.

Use official platform documentation first. Supplement with trusted industry sources when necessary. Cite every source.

## Phase 5 — AI Recommendation Rules

* Which image types should this brand create?
* Which platforms should they prioritize?
