# Research Task: Design the AI Campaign Recommendation Engine

## Objective

Design an expert AI Campaign Strategist that helps brands create high-performing marketing campaigns using the best combination of images, videos, ads, ecommerce assets, and social media content.

The AI should behave like a senior Creative Director, Ecommerce Manager, Social Media Manager, Performance Marketer, and Brand Strategist.

The AI should recommend what to create, why, when, where to publish it, and how to maximize results.
