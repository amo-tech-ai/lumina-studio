# Research Mission

Build the world's most comprehensive Social Media Image Intelligence Database.

This will become a permanent knowledge base used by AI agents inside FashionOS/iPix.

The AI must be able to intelligently recommend:

• which image types to create
• which sizes
• which aspect ratios
• which file formats
• which resolution
• when each image should be used
• why
• expected business goals
• estimated effectiveness
• channel compatibility
• campaign recommendations

This is NOT a simple image size guide.

This is an enterprise knowledge base.

Use official documentation whenever possible.

When official documentation is unavailable use respected industry sources.

Always cite sources.

Never guess.
