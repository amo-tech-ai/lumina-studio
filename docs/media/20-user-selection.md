# Research Task: Design an Intelligent Media Booking & Asset Selection System

## Objective

Design an AI-powered booking system that helps customers select the right images, videos, and media assets for their project.

The AI should guide users through the booking process like an experienced Creative Director, Photographer, Marketing Strategist, and Ecommerce Consultant.

The system should recommend the most effective media based on the customer's goals, industry, budget, and marketing objectives.
