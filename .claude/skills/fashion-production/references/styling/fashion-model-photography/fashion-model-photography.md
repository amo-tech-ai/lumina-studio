# Fashion Model Photography — Real Shoot Direction

## When to Use This Skill

Use when you need to:
- Create model direction cards for a real photographer
- Write casting briefs to send to model agencies
- Build stylist pull lists and wardrobe plans
- Define HMU (hair/makeup) direction per brand DNA
- Specify pose vocabulary for on-set direction
- Build the talent section of a production package

## Core Principle

**This skill produces documents for real humans on set — photographers, models, stylists, makeup artists.** It does NOT generate AI images or call image-generation APIs. Every output is a brief, card, or checklist that a person reads.

## Document Outputs

This skill generates 5 documents:

1. **Casting Brief** — What to send the model agency
2. **Model Direction Card** — On-set pose/expression guidance (printable)
3. **Stylist Pull List** — Wardrobe coordination document
4. **HMU Brief** — Hair and makeup direction
5. **Photographer Direction** — Camera, lighting, composition per setup

## 1. Casting Brief

Send to model agency to book the right talent.

```
CASTING BRIEF — [Brand Name] [Campaign Name]

SHOOT DATE: [Date]
LOCATION:   [Studio/Location]
RATE:       [Day rate / half-day / hourly]
USAGE:      [Ecommerce / Social / Editorial / Campaign — affects rate]

LOOKING FOR:
  Gender:     [Male / Female / Non-binary / Any]
  Age range:  [18-25 / 25-35 / 35-45 / etc.]
  Body type:  [Per brand requirement — inclusive range encouraged]
  Height:     [If garment-specific — e.g., sample size requirement]
  Skin tone:  [Describe range — "warm medium to deep" not a single shade]
  Hair:       [Open / specific needs per styling concept]

BRAND AESTHETIC:
  [2-3 sentences describing the brand's visual identity]
  Reference: [Brand URL for agency to review]
  Mood:      [Editorial / Commercial / Natural / Luxury / Streetwear]

WARDROBE:
  Sample size: [Specific size — models must fit samples]
  Product type: [Garments / Accessories / Jewelry / Footwear]
  Note:        [Any requirements — e.g., "hands must be elegant for jewelry"]

WHAT WE NEED:
  □ Digitals (no makeup, natural lighting)
  □ Portfolio book
  □ Comp card
  □ Availability for [date] + potential reshoot [date]

SUBMIT TO: [Contact name, email, deadline]
```

### Pose Vocabulary — On-Set Reference

**Body Position:**

| Direction | Meaning | When to Use |
|-----------|---------|-------------|
| "Weight shift left" | Left leg carries weight, right relaxed | Standing poses, creates S-curve |
| "Open your frame" | Angle body 30° to camera | Most editorial — avoids flat front |
| "Walk to me" | Model walks toward camera | Movement shots, natural stride |
| "Give me the turn" | Model mid-turn, looking back | Dynamic energy, shows garment drape |
| "Hands live" | Hands doing something natural | Avoid dead hands at sides |
| "Drop your shoulders" | Relax tension in shoulders | Fixes stiff posture |

**Expression:**

| Direction | Meaning | Result |
|-----------|---------|--------|
| "Soft eyes" | Relax brow, gentle gaze | Luxury, beauty, editorial |
| "Think past me" | Look at something beyond camera | Contemplative, editorial |
| "You just saw someone you know" | Natural recognition response | Warm, genuine smile |
| "Own the room" | Confidence without aggression | Power, campaign imagery |
| "Mid-breath" | Capture between poses | Most authentic moments |

**Product-Specific Direction:**

| Product Type | Direction | Why |
|-------------|-----------|-----|
| Garments | "Show me the drape" — move so fabric flows | Fabric movement sells quality |
| Jewelry | "Lead with the hand" — gesture that displays piece | Focus on product, not just face |
| Bags | "It's your favorite thing" — hold naturally, not displayed | Authenticity > catalog display |
| Footwear | "Walk past me, pause, look back" — shows shoe in motion | Shoes need to be worn, not just shown |
| Eyewear | "Chin slightly down, eyes up through the frames" | Frames visible, eyes engaged |

```
  Look 2 — [Mood: weekend casual]
    □ Bottom: [e.g., light wash denim, straight leg, cropped]

    □ Accessory: [e.g., canvas tote, natural color]
    □ Avoid: [e.g., athletic wear, neon, distressed denim]

  Look 3 — [Mood: editorial dramatic]
    □ Bottom: [e.g., black leather trouser, high waist]
    □ Shoe: [e.g., pointed heel, black, 3" minimum]
    □ Accessory: [e.g., statement earring, gold, geometric]
    □ Avoid: [e.g., casual elements, flats, small prints]

BRAND DNA COLOR PALETTE:
  Primary:   [Color] — hero pieces
  Secondary: [Color] — supporting pieces
  Neutral:   [Color] — basics, foundations
  Avoid:     [Colors that clash with brand DNA]

FABRIC GUIDELINES:
  Prefer: [Silk, cashmere, linen, quality cotton — per brand tier]
  Avoid:  [Polyester sheen, cheap jersey, pilling fabrics]
  Note:   [Steam all garments before set. No visible tags.]

FITTING NOTES:
  □ All looks fitted to model before shoot day
  □ Clips/pins on back for camera-side fit adjustments
  □ Backup options for each look (in case of fit issues)
  □ Steamer available on set
```

## 4. HMU Brief

```
HMU BRIEF — [Brand] [Campaign]

BRAND DNA: [Warm/Cool/Neutral — Luxury/Modern/Natural]
REFERENCE: [Brand URL for visual reference]

MAKEUP DIRECTION:

  Level: [Natural / Elevated Natural / Editorial]

  Skin:
    Base:      [Light coverage, skin-like finish]
    Texture:   [Visible pores — do NOT over-smooth]
    Glow:      [Natural luminosity, not glitter]
    Note:      [iPix brands value real skin texture]

  Eyes:
    Shadow:    [Neutral tones / smoky / none]
    Liner:     [Tight-line only / wing / none]
    Lashes:    [Mascara only / individual false / strip]
    Brows:     [Natural, groomed / defined, filled]

  Lips:
    Color:     [Nude-rose / berry / bold red / your-lips-but-better]
    Finish:    [Matte / satin / gloss]

  Contour:
    Level:     [Minimal / Moderate — no heavy Instagram contour]
    Note:      [Enhance natural bone structure, not reshape]

HAIR DIRECTION:

  Style:
    [e.g., "Natural texture, volume at crown, off the face"]
    [e.g., "Sleek low bun, center part, face-framing pieces"]
    [e.g., "Loose waves, side part, movement on ends"]

  Product-Specific:
    Earrings → Hair up or behind ears (jewelry visible)
    Necklaces → Hair up or swept to one side (neckline clear)
    Headwear → Style around the piece (no competing volume)
    High neckline → Hair up (show garment detail)

  Avoid:
    □ Over-styled, "done" look (unless editorial)
    □ Flyaways breaking silhouette excessively
    □ Product in hair (no visible clips unless intentional)

TOUCH-UP KIT ON SET:
  □ Blotting papers
  □ Setting spray
  □ Lip color for reapplication
  □ Bobby pins matching hair color
  □ Steamer (hair refresh between setups)

TIMING:
  HMU call time: [Time — typically 1-1.5h before first shot]
  Look changes:  [Time between looks — 15-30 min per change]
```

## 5. Photographer Direction

```
PHOTOGRAPHER DIRECTION — Fashion/Editorial Setups

GENERAL:
  Tethered shooting: YES (always for fashion — check focus on monitor)
  Shoot cadence:     3-5 frames per pose variation
  Direction style:   Talk through poses (don't just say "next")
  Energy:            Music on set matches brand mood

SETUP A — [Product/Campaign Name]:
  Camera:   [Full-frame / medium format]
  Lens:     [85mm for portraits / 50mm for 3/4 / 35mm for full body+environment]
  Aperture: [f/2.8 for isolation / f/4-5.6 for balance / f/8 for everything sharp]
  Angle:    [Eye-level / slightly low for power / slightly high for vulnerability]

  Lighting: [See lighting-plan-generator for full setup]

  Shoot flow:
    1. Start with safe shot (centered, neutral pose)
    2. Move to direction (guide into poses from direction cards)
    3. Capture transitions (best shots happen between poses)
    4. Review tethered every 5-10 shots (exposure/focus check)
    5. Adjust based on what works (adapt, don't over-plan)

  Must-get shots:
    □ [Specific shot list reference — e.g., "A01-A05 per shot list"]
    □ [Any must-have angles for specific channels]

  Creative freedom:
    □ [What the photographer can experiment with]
    □ [Boundaries — what NOT to do]
```

## Key Principles (Real Photography)

1. **Brief, Don't Generate** — Every output is a document for a human
2. **Authenticity Over Perfection** — Mid-breath > rigid pose
3. **Product First** — Composition led by product visibility, not model beauty
4. **Real Skin Texture** — NEVER specify over-smoothed/poreless skin
5. **Brand DNA Match** — Every direction aligns with brand's visual identity
6. **Printable Cards** — Direction cards must work printed on set
7. **Adaptable** — Photographers adapt on set. Briefs guide, they don't restrict.

## Validation

- [ ] Casting brief has all fields for agency submission
- [ ] Direction cards are printable (one page each, clear formatting)
- [ ] Stylist pull list has specific items (not vague descriptors)
