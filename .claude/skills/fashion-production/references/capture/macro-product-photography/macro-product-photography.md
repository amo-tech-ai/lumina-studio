# Macro Product Photography — Detail Shot QC Standards

## When to Use This Skill

Use when you need to:
- Define what detail shots a photographer must capture
- Specify macro equipment and settings per material type
- Create QC pass/fail criteria for detail shot uploads
- Brief AestheticAuditor on what "good" looks like per material
- Build the detail shot section of a production package

## Core Principle

**ACCURACY > AESTHETICS.** Detail shots document quality and craftsmanship. If stitching is uneven in reality, the photo must show that — it's QC, not marketing. These shots prove product quality to the customer.

## Three-Level Material Specificity

Every detail shot must identify the material at three levels:

| Level | What It Answers | Example |
|-------|----------------|---------|
| **Level 1 — Base** | What is it made of? | Leather |
| **Level 2 — Construction** | How is it made/structured? | Full-grain, vegetable-tanned |
| **Level 3 — Surface** | What does the surface look/feel like? | Smooth with slight natural pebble, matte |

### Material Spec Examples

```
CORRECT (three-level):
  "Cotton canvas weave with matte, medium-textured finish"
  "Full-grain leather with smooth, slightly pebbled finish"
  "Brushed nickel metal with linear matte finish"
  "Silk charmeuse with high-sheen, fluid drape"

WRONG (too vague):
  "Fabric" / "Leather" / "Metal" / "Silk"
```

## Detail Shot Checklist by Product Category

### Garments

```
GARMENT DETAIL SHOTS (minimum 5 per product):

1. STITCHING — Primary seam
   What:     Top-stitch at collar, shoulder, or side seam
   Focus on: Thread color, stitch count per inch, needle lines (single/double)
   Spec:     100mm macro, f/8, 30° angle, tripod
   QC:       Even stitch spacing? Consistent tension? No skipped stitches?

2. FABRIC TEXTURE — Main body material
   What:     Weave pattern, surface texture, fiber visibility
   Focus on: Three-level material spec clearly visible
   Spec:     100mm macro, f/11, 45° angle (show dimension), tripod
   QC:       Weave direction correct? Color accurate to physical sample?

3. HARDWARE — Buttons, zippers, clasps
   What:     Close-up of each hardware piece
   Focus on: Material finish, branding (if present), mechanism
   Spec:     100mm macro, f/8, ring light for even illumination
   QC:       Finish consistent? No scratches? Logo legible (if branded)?

4. LABEL/TAG — Brand identification
   What:     Main label, care label, size tag
   Focus on: Print quality, stitching attachment, alignment
   Spec:     100mm macro, f/8, flat lighting
   QC:       Text legible? Centered? Stitching clean?

5. CONSTRUCTION — Seam/hem detail
   What:     Inside seam finishing, hem construction, lining attachment
   Focus on: Seam type (French, overlock, flat-felled), thread finish
   Spec:     100mm macro, f/8-f/11, 30° angle
   QC:       Clean finishing? No loose threads? Consistent construction?

OPTIONAL (per product):
  6. Lining detail — if premium lining is a selling point
  7. Pocket construction — if pocket detail matters
  8. Embroidery/print — if decorative elements present
```

### Accessories (Bags, Wallets, Belts)

```
ACCESSORY DETAIL SHOTS (minimum 5 per product):

1. MATERIAL — Primary surface
   What:     Leather grain, fabric weave, surface texture
   Focus on: Three-level material spec
   Spec:     100mm macro, f/8-f/11, 45° angle

2. HARDWARE — Clasps, buckles, zippers, studs
   What:     Each unique hardware piece
   Focus on: Metal type, finish, mechanism, branding
   Spec:     100mm macro, f/8, ring light
   QC:       Finish consistent across all hardware? Color match?

3. STITCHING — Edge stitching, handle attachment
   What:     Stitching at stress points (handle, strap, closure)
   Focus on: Thread quality, stitch density, contrast
   Spec:     100mm macro, f/8, 30° angle

4. EDGE FINISHING — Leather edge, zipper tape, binding
   What:     How edges are finished (burnished, painted, raw, folded)
   Focus on: Smoothness, color consistency, craftsmanship
   Spec:     100mm macro, f/11, profile angle (90° to edge)

5. INTERIOR — Lining, pockets, organization
   What:     Inside construction, lining material, pocket detail
   Focus on: Finish quality, logo stamp, card slots
   Spec:     100mm macro, f/5.6-f/8 (working space)
```

### Jewelry

```
JEWELRY DETAIL SHOTS (minimum 4 per product):

1. SETTING — Stone/gem mounting
   What:     How stones are set (prong, bezel, channel, pave)
   Focus on: Setting precision, stone alignment, metal work
   Spec:     100mm macro + extension tube (high magnification)
            Focus stacking required (2-5 frames)
            f/11, tripod, ring light

2. CLASP/CLOSURE — Functioning mechanism
   What:     Lobster claw, toggle, spring ring, magnetic
   Focus on: Mechanism quality, finish, ease of use
   Spec:     100mm macro, f/8, multiple angles (open + closed)

3. HALLMARK/STAMP — Metal purity, brand mark
   What:     Stamped marks (925, 750, brand logo)
   Focus on: Legibility, depth of stamp, location
   Spec:     100mm macro + extension tube, f/11, directional light

4. SURFACE FINISH — Metal texture
   What:     Polish, matte, brushed, hammered, oxidized
   Focus on: Reflectivity, texture pattern, consistency
   Spec:     100mm macro, f/8, controlled reflections (tent lighting)
```

### Footwear

```
FOOTWEAR DETAIL SHOTS (minimum 5 per product):

1. SOLE — Construction and material
   What:     Sole material, tread pattern, branding
   Focus on: Sole type (leather, rubber, TPU), stitching to upper
   Spec:     100mm macro, f/8, 45° angle from below

2. UPPER MATERIAL — Primary material
   What:     Leather grain, fabric weave, suede nap
   Focus on: Three-level material spec
   Spec:     100mm macro, f/11, 30° angle

3. STITCHING — Welt, upper to sole, decorative
   What:     Construction stitching (Blake, Goodyear, cemented)
   Focus on: Stitch quality, thread color, construction method
   Spec:     100mm macro, f/8, profile angle

4. HARDWARE — Eyelets, buckles, zippers
   What:     Each unique hardware piece
   Focus on: Material, finish, function
   Spec:     100mm macro, f/8, ring light

5. INSOLE — Interior comfort, branding
   What:     Insole material, brand stamp, cushioning
   Focus on: Print quality, material quality
   Spec:     100mm macro, f/5.6-f/8
```

## Equipment Requirements

```
MACRO PHOTOGRAPHY KIT:

Camera:
  □ Full-frame DSLR or mirrorless (high resolution sensor)
  □ Minimum 24MP (for crop flexibility)

Lenses:
  □ 100mm macro (primary — 1:1 magnification)
  □ Extension tubes (for higher than 1:1 if needed — jewelry)
  □ 50mm standard (for context shots)

Support:
  □ Tripod (mandatory — no handheld macro)
  □ Focus rail (precise forward/back focus adjustment)
  □ Remote shutter release (cable or wireless)
  □ L-bracket (for vertical orientation on tripod)

Lighting:
  □ Ring light (even, shadowless for small subjects)
  □ Small softbox 30-40cm (directional for texture)
  □ LED panel (continuous for focusing assistance)
  □ White/silver reflector cards (fill shadows)
  □ Diffusion material (soften harsh light)

Accessories:
  □ Grey card (color calibration each session)
  □ Compressed air (dust removal before shooting)
  □ Lint roller (remove threads from fabric)
  □ Tweezers (remove stray threads)
  □ Cleaning cloth (fingerprints on hardware)
  □ Scale ruler (for sizing reference if needed)
```

## QC Standards Table

| Material Type | Pass Criteria | Fail Criteria |
|--------------|--------------|---------------|
| Stitching | Even spacing, consistent tension, no skips | Uneven gaps, loose threads, missed stitches |
| Leather | True color, visible grain at spec level, no cracks | Color cast, grain invisible, surface damage |
| Metal hardware | Finish consistent, no scratches, mechanism works | Scratches visible, uneven finish, tarnish |
| Fabric weave | Pattern clear, color accurate, no pilling | Pattern blurred, color off, visible defects |
| Edge finish | Smooth, consistent color, no gaps | Rough, uneven color, peeling |
| Print/logo | Legible, centered, consistent color | Blurred, off-center, fading |

## Common Macro Mistakes

| Mistake | Why It Happens | Fix |
|---------|---------------|-----|
| Soft focus | Handheld shooting, wrong focus point | Use tripod + focus rail. Manual focus with peaking. |
| Flat perpendicular angle | Shot straight-on, no dimension | Angle camera 20-45° to show texture depth |
| Harsh shadows | Light too directional/small | Use ring light or large soft source close to subject |
| Color cast | Mixed lighting, no calibration | Grey card at start of session, match all lights to same temp |
| Over-sharpening in post | Trying to fix soft capture | Get it sharp in-camera. Sharpening can't fix soft focus. |
| Missed details | No checklist, shooting randomly | Follow category checklist above — systematic, not ad hoc |
| Dust/threads visible | Didn't clean product before shooting | Compressed air + lint roller BEFORE every detail shot |

## Integration with AestheticAuditor

```
DETAIL SHOT QC FLOW:

Photographer shoots detail checklist
  ↓
Upload to iPix (organized by asset-triage skill)
  ↓
AestheticAuditor receives detail shots
  ↓
Scores against three-level material spec:
  - Is Level 1 (base material) clearly identifiable? (20 pts)
  - Is Level 2 (construction) visible? (30 pts)
  - Is Level 3 (surface finish) accurately represented? (30 pts)
  - Is the shot technically sound? (20 pts — focus, exposure, angle)
  ↓
Score < 80 → Retake flagging skill generates fix instruction
Score ≥ 80 → Pass → Ready for post-production
```
